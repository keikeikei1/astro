Markdown 是一种轻量级标记语言，让您可以使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

## 基本语法

- **标题**：使用 `#` 符号，例如 `# 一级标题`
- **粗体**：使用 `**文本**` 或 `__文本__`
- **斜体**：使用 `*文本*` 或 `_文本_`
- **链接**：`[链接文本](URL)`
- **图片**：`![替代文本](图片URL)`
- **列表**：使用 `-` 或 `*` 或 `1.` 创建列表
- **引用**：使用 `>` 符号

这些简单的语法可以帮助您快速创建格式丰富的文档，而无需使用复杂的编辑器。
